# Introduction
 This application creates and manages a household task list for all members of a given household. Each member of a household has their own account that can be used to introduce new tasks to the household, complete tasks, and edit/give feedback on tasks. 
 Households survive on a list of repeatable, expected tasks, along with last-minute additions to the house's responsibilities. All need to be managed in such a way that each member of the house should own their fair share of responsibilities. 

 # Design Guidelines 
Using this app should feel like writing on a whiteboard or adding sticky notes to a taskboard. The user experience should be friendly to motivate users to complete their tasks. Completing a task should feel rewarding to the user who completed it. To create additional moments of delight, use motion and animation to add a bit of magic to the experience.

 # Key UIs to create 
 * Splash page with information about the application and what it can do
 * My household's task list, with a summary view of every upcoming task that each member of the house needs to complete
 * My task list
 * Idea generator: if my household has gotten too efficient at completing tasks, give them some more challenging tasks to complete and fill their day
